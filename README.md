# 📋 ReqAI – AI Requirements Generator

> An AI-powered tool that transforms business problems into structured requirements — user stories, acceptance criteria, and process flows — in seconds.

---

## 🎯 Why I Built This

As a Business Analyst in financial services, I've seen how much time gets spent translating vague feature requests into structured, sprint-ready requirements. This tool automates the first draft — giving BAs, product owners, and developers a solid starting point they can refine rather than build from scratch.

Built as a personal project to explore practical AI application in BA workflows.

---

## ✨ Features

- **Instant requirements generation** — paste in a business problem and get structured output in seconds
- **Three output types** — User Stories, Acceptance Criteria, and Process Flow
- **Flexible formats** — supports Agile/Scrum, BDD (Given/When/Then), and Waterfall
- **Priority modes** — MVP/Core, Full Feature Set, or Edge Cases focus
- **Domain-aware** — tailored prompting for Financial Services, Insurance, Healthcare, and more
- **Copy-ready output** — each section has a one-click copy button for pasting into Jira, Confluence, or Word

## 💡 Example Use Case

**Input:**
> Advisers are manually emailing referrals to lending firms, causing delays and data errors. We need a digital portal for introducer firms to submit referrals directly.

**Output includes:**
- 3-4 structured user stories
- 6-8 testable acceptance criteria
- 6-8 step process flow
